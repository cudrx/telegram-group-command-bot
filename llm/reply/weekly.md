Составь недельный дайджест чата по предоставленному WEEKLY_DATASET.

Пиши по-русски. Формат вывода: Telegram HTML. Разрешены только теги <b>, <i>, <code>. Не используй Markdown, списки с дефисами, таблицы, ссылки без явной опоры в данных или выдуманные цитаты.

Используй только факты из WEEK_STATS, PARTICIPANT_STATS и SELECTED_EVENTS. Если SELECTED_EVENTS пустой или слабый, честно сделай короткий спокойный обзор по статистике, без драматизации. Не называй пользователей по userId, если есть displayName. Не раскрывай внутренние поля, оценки score, messageIds, participantIds или причины отбора как технические данные.

Обязательная структура:

<b>Неделя в чате</b>

• 2-3 пункта с главным ходом недели.

<b>Темы</b>

• Только темы, подтвержденные SELECTED_EVENTS.

<b>Моменты</b>

• Конкретные эпизоды недели, не общий пересказ настроения.

<b>Роли недели</b>

• Только роли, заметные по сообщениям этой недели.

<b>Факты</b>

• Короткие факты с опорой на данные недели.

<b>Итог</b>

Один короткий завершающий абзац.
