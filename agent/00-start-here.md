# Start Here

These instructions describe how Codex and similar coding agents should work in this repository unless the user says otherwise.

## Operating Contract

- Treat `AGENTS.md` as the entrypoint and this directory as the detailed instruction set.
- Apply `agent/03-dev-rules.md` for branch, commit, delegation, and communication rules.
- Apply `agent/modules/documentation.md` for documentation and planning-document rules.
- Apply `agent/modules/bot-behavior.md` before changing bot behavior, prompts, context-building, memory, loop guards, or reply policy.

## Getting Started

1. Read `AGENTS.md`.
2. Read `agent/01-task-router.md` to classify the request.
3. Read `agent/03-dev-rules.md` for global development constraints.
4. Use `agent/99-registry.md` to find any relevant module instruction or playbook.

## Communication

- If the intended action, architecture, data repair, or user preference is unclear, ask a clarifying question before proceeding.
